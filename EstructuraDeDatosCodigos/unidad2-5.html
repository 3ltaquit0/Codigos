Explicación del Código

El siguiente código en Java utiliza un método recursivo para invertir una palabra ingresada por el usuario.

Clase Recursividad


import java.util.Scanner;

public class Recursividad {
    static String invierte(String palabra) {
        if (palabra.isEmpty()) {
            return palabra;
        } else {
            return invierte(palabra.substring(1)) + palabra.charAt(0);
        }
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Ingresa una palabra: ");
        String palabra = scanner.nextLine();

        System.out.println("Palabra invertida: " + Recursividad.invierte(palabra));

        scanner.close();
    }
}

Explicación

La clase Recursividad contiene dos métodos principales:

  • invierte(String palabra): Este método recursivo invierte la palabra dada, moviendo el primer carácter al final y llamándose recursivamente con el resto de la palabra.
  • main(String[] args): Este método es el punto de entrada del programa, que solicita al usuario una palabra y muestra la palabra invertida.

Método invierte

El método invierte sigue estos pasos:

  1. Comprueba si la palabra está vacía. Si es así, retorna la palabra vacía (caso base).
  2. Si no, toma el primer carácter de la palabra y lo coloca al final de la palabra invertida resultante de llamar recursivamente al método con el resto de la palabra (todo menos el primer carácter).

Método main

El método main realiza lo siguiente:

  1. Pide al usuario que ingrese una palabra.
  2. Invoca el método invierte con la palabra ingresada y muestra la palabra invertida en la consola.
  3. Cierra el escáner para liberar recursos.
