Explicación del Algoritmo Bubble Sort

Este código en Java implementa el algoritmo de Bubble Sort, que ordena una lista de números de menor a mayor. A continuación se explica cómo funciona el código y el rendimiento de Bubble Sort en distintos tamaños de listas. 

import java.util.Random;

public class BubbleSort {

    // Método para ordenar un arreglo usando el algoritmo de Bubble Sort
    public static void bubblesort(int[] arr) {
        int n = arr.length;
        // Recorre todo el arreglo
        for (int i = 0; i < n - 1; i++) {
            for (int j = 0; j < n - 1; j++) {
                // Si el elemento actual es mayor que el siguiente, intercámbialos
                if (arr[j] > arr[j + 1]) {
                    int temp = arr[j];
                    arr[j] = arr[j + 1];
                    arr[j + 1] = temp;
                }
            }
        }
    }

    // Método para mostrar el arreglo ordenado
    public static void printArray(int[] arr) {
        for (int i : arr) {
            System.out.print(i + " ");
        }
        System.out.println();
    }

    public static void main(String[] args) {
        int[] sizes = {10, 100, 1000, 5000};
        Random random = new Random();

        for (int size : sizes) {
            int[] arr = new int[size];
            for (int i = 0; i < size; i++) {
                arr[i] = random.nextInt(10000);
            }

            long startTime = System.currentTimeMillis();
            bubblesort(arr);
            long endTime = System.currentTimeMillis();

            System.out.println("Tiempo para ordenar lista de tamaño " + size + ": " + (endTime - startTime) + " ms");
            printArray(arr);
        }
    }
}

Explicación del Código

El código tiene tres partes principales: el método bubblesort para ordenar, el método printArray para mostrar el arreglo y el método main que ejecuta el proceso y mide el tiempo de ordenación.

Método bubblesort()

Este método recorre el arreglo y compara cada elemento con el siguiente. Si el elemento actual es mayor, se intercambian. Este proceso se repite hasta que todos los elementos están en orden ascendente.

Método printArray()

El método printArray() imprime los elementos del arreglo ordenado, mostrando el resultado final de la ordenación.

Método main()

El método main crea arreglos de diferentes tamaños, los llena con valores aleatorios, y mide el tiempo que toma ordenarlos con bubblesort. Esto permite observar el rendimiento del algoritmo.
