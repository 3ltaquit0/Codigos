Explicación del Código

El siguiente código en Java utiliza un método recursivo para calcular el Máximo Común Divisor (MCD) de dos números.

Clase Metodo1


package Recursividad;

//Método recursivo para calcular máximo común divisor de 2 números

public class Metodo1 {
  public static int mcd(int a, int b) {
      if (b == 0) {
          return a;
      } else {
          return mcd(b, a % b);
      }
  }

  public static void main(String[] args) {
      System.out.println(mcd(10, 5));
  }
}

Explicación

La clase Metodo1 contiene dos métodos principales:

  • mcd(int a, int b): Este método recursivo calcula el máximo común divisor de dos números utilizando el algoritmo de Euclides.
  • main(String[] args): Este método es el punto de entrada del programa y prueba el cálculo del MCD de dos números.

Método mcd

El método mcd sigue estos pasos:

  1. Si el segundo número (b) es 0, retorna el primer número (a) como el MCD.
  2. Si no, realiza una llamada recursiva con los valores de b y a % b, donde % es el operador módulo.

Método main

El método main realiza lo siguiente:

  1. Invoca el método mcd con dos números (en este caso, 10 y 5) y muestra el resultado en la consola.
