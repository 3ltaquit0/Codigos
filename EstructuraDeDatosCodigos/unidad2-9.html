Explicación del Código

Este programa en Java cuenta cuántas veces aparece un carácter específico dentro de una cadena de texto utilizando un enfoque recursivo.

Clase ContadorDeCaracteres


import java.util.Scanner;

public class ContadorDeCaracteres {

    // Método recursivo para contar el número de ocurrencias de un carácter
    public static int contarCaracter(char caracter, String cadena, int indice) {
        if (indice == cadena.length()) {
            return 0;
        }
        int conteo = (cadena.charAt(indice) == caracter) ? 1 : 0;
        return conteo + contarCaracter(caracter, cadena, indice + 1);
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Ingrese una cadena de texto: ");
        String cadena = scanner.nextLine();

        System.out.print("Ingrese un carácter a contar: ");
        char caracter = scanner.next().charAt(0);

        System.out.println("El carácter '" + caracter + "' aparece " +
                           contarCaracter(caracter, cadena, 0) + " veces en la cadena.");

        scanner.close();
    }
}

Explicación

La clase ContadorDeCaracteres implementa dos métodos principales:

  • contarCaracter(char caracter, String cadena, int indice): Este método recursivo toma un carácter, una cadena de texto, y un índice, y cuenta cuántas veces aparece el carácter en la cadena desde el índice proporcionado hasta el final.
  • main(String[] args): Este método es el punto de entrada del programa. Solicita al usuario que ingrese una cadena de texto y un carácter, y luego muestra cuántas veces aparece el carácter en la cadena.

Método contarCaracter

El método contarCaracter sigue estos pasos:

  1. Verifica si el índice actual es igual a la longitud de la cadena, en cuyo caso la recursión termina y retorna 0 (caso base).
  2. Compara el carácter actual de la cadena con el carácter a contar. Si coinciden, suma 1 al resultado, de lo contrario suma 0.
  3. Llama recursivamente al método para el siguiente índice (indice + 1), acumulando el número de coincidencias.

Método main

El método main realiza lo siguiente:

  1. Solicita al usuario que ingrese una cadena de texto.
  2. Pide al usuario que introduzca el carácter que desea contar.
  3. Utiliza el método contarCaracter para contar cuántas veces aparece el carácter en la cadena, comenzando desde el índice 0.
  4. Muestra el resultado al usuario.
