Explicación del Código

El siguiente código en Java utiliza un método recursivo para invertir un número sin tratarlo como una cadena.

Clase InvertirNumero


public class InvertirNumero {

    //programa que invierte un número pero que se maneje como número y no como cadena

    static int invertirN(int numero){
        if (numero < 10) return numero;
        else {
            int digito = numero % 10;
            int longitud = (int) Math.log10(numero);
            return digito * (int) Math.pow(10, longitud) + invertirN(numero / 10);
        }
    }

    public static void main(String[] args){
        System.out.println(invertirN(1234));
    }
}

Explicación

La clase InvertirNumero contiene dos métodos principales:

  • invertirN(int numero): Este método recursivo invierte el número dado utilizando operaciones matemáticas en lugar de convertirlo a una cadena.
  • main(String[] args): Este método es el punto de entrada del programa, que toma un número entero y lo invierte.

Método invertirN

El método invertirN sigue estos pasos:

  1. Si el número es menor que 10, se devuelve tal cual (caso base).
  2. En el caso contrario:
    • Calcula el último dígito del número usando el módulo % 10.
    • Calcula la longitud del número usando Math.log10() para encontrar cuántos dígitos tiene.
    • Multiplica el dígito extraído por Math.pow(10, longitud) para colocarlo en la posición invertida.
    • Llama recursivamente al método con el número restante (numero / 10).

Método main

El método main realiza lo siguiente:

  1. Ejecuta el método invertirN con un número de prueba, como 1234.
  2. Muestra el número invertido en la consola.
