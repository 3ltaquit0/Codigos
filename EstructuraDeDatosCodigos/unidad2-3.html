Explicación del Código

El siguiente código en Java verifica si una palabra es un palíndromo utilizando un método recursivo.

Clase Palindromo


import java.util.Scanner;

public class Palindromo {

    static boolean esPalindromo(String palabra) {

        if (palabra.length() <= 1) {
            return true;
        }

        if (palabra.charAt(0) == palabra.charAt(palabra.length() - 1)) {
            return esPalindromo(palabra.substring(1, palabra.length() - 1));
        }

        return false;
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);

        System.out.print("Ingresa una palabra: ");
        String palabra = scanner.nextLine();

        palabra = palabra.replaceAll("\\s+", "").toLowerCase();

        if (esPalindromo(palabra)) {
            System.out.println("La palabra es un palíndromo.");
        } else {
            System.out.println("La palabra no es un palíndromo.");
        }

        scanner.close();
    }
}

Explicación

La clase Palindromo contiene dos métodos principales:

  • esPalindromo(String palabra): Este método recursivo verifica si una palabra es un palíndromo, comparando el primer y el último carácter, y recortando la palabra en cada paso.
  • main(String[] args): Este método es el punto de entrada del programa, donde se solicita al usuario que ingrese una palabra, y se determina si es un palíndromo o no.

Método esPalindromo

El método esPalindromo sigue estos pasos:

  1. Si la palabra tiene 0 o 1 caracteres, retorna true, indicando que es un palíndromo.
  2. Compara el primer carácter con el último.
  3. Si son iguales, realiza una llamada recursiva con la palabra reducida, eliminando el primer y el último carácter.
  4. Si los caracteres no coinciden, retorna false.

Método main

El método main realiza lo siguiente:

  1. Pide al usuario que ingrese una palabra a través de la consola.
  2. Elimina los espacios y convierte la palabra a minúsculas para evitar diferencias de formato.
  3. Verifica si la palabra es un palíndromo utilizando el método esPalindromo y muestra el resultado en la consola.
