Explicación del Código

El siguiente código en Java verifica si los paréntesis en una ecuación están balanceados utilizando una pila.

Clase ValidadorParentesis


package Pila;

public class ValidadorParentesis {

    public static boolean estaBalanceado(String ecuacion) {
        Pila pila = new Pila(ecuacion.length());

        for (int i = 0; i < ecuacion.length(); i++) {
            char caracter = ecuacion.charAt(i);

            if (caracter == '(') {
                pila.push(caracter);
            } else if (caracter == ')') {
                if (pila.estaVacia()) {
                    return false;
                }
                pila.pop();
            }
        }

        return pila.estaVacia();
    }

    public static void main(String[] args) {
        String ecuacion1 = "2x+(5y*3)-4";
        String ecuacion2 = "(a + b * (c + d)";
        String ecuacion3 = "(a + b) * c + d)";

        System.out.println("Ecuacion 1 balanceada: " + estaBalanceado(ecuacion1));
        System.out.println("Ecuacion 2 balanceada: " + estaBalanceado(ecuacion2));
        System.out.println("Ecuacion 3 balanceada: " + estaBalanceado(ecuacion3));
    }
}

Explicación

La clase ValidadorParentesis contiene dos métodos principales:

  • estaBalanceado(String ecuacion): Este método verifica si los paréntesis en la ecuación están balanceados. Utiliza una pila para realizar esta verificación.
  • main(String[] args): Este método es el punto de entrada del programa. Prueba varias ecuaciones y muestra si están balanceadas o no.

Método estaBalanceado

El método estaBalanceado sigue estos pasos:

  1. Crea una pila con la capacidad igual a la longitud de la ecuación.
  2. Recorre cada carácter de la ecuación.
  3. Si el carácter es un paréntesis de apertura '(', lo empuja a la pila.
  4. Si el carácter es un paréntesis de cierre ')', verifica si la pila está vacía. Si está vacía, retorna false. Si no está vacía, saca un elemento de la pila.
  5. Al final del recorrido, retorna true si la pila está vacía, lo que significa que los paréntesis están balanceados.

Método main

El método main realiza lo siguiente:

  1. Define tres ecuaciones de prueba.
  2. Verifica si cada ecuación está balanceada utilizando el método estaBalanceado.
  3. Muestra el resultado en la consola.
