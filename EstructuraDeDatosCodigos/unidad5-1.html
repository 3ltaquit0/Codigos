Explicación del Código de Altura y Profundidad en Árbol Binario

Este código en Java crea un árbol binario simple y calcula la altura del árbol y la profundidad de un nodo específico. A continuación se explica cómo funciona cada parte del código. 

public class AlturaArbolBinario {
    // Clase interna que representa un nodo del árbol binario
    class Nodo {
        int dato; // Valor del nodo
        Nodo izquierdo, derecho; // Referencias a los nodos hijo izquierdo y derecho

        public Nodo(int item) {
            dato = item;
            izquierdo = derecho = null;
        }
    }

    Nodo raiz;

    // Método para calcular la altura de un nodo en el árbol
    int altura(Nodo nodo) {
        if (nodo == null) {
            return 0;
        } else {
            int alturaIzquierda = altura(nodo.izquierdo);
            int alturaDerecha = altura(nodo.derecho);
            return Math.max(alturaIzquierda, alturaDerecha) + 1;
        }
    }

    // Método para calcular la profundidad de un nodo en el árbol
    int profundidad(Nodo nodo, Nodo raiz) {
        if (raiz == null) {
            return -1;
        }
        if (raiz == nodo) {
            return 0;
        }
        int nivel = profundidad(nodo, raiz.izquierdo);
        if (nivel == -1) {
            nivel = profundidad(nodo, raiz.derecho);
        }
        if (nivel != -1) {
            return nivel + 1;
        }
        return -1;
    }

    public static void main(String[] args) {
        AlturaArbolBinario arbol = new AlturaArbolBinario();
        arbol.raiz = arbol.new Nodo(1);
        arbol.raiz.izquierdo = arbol.new Nodo(2);
        arbol.raiz.derecho = arbol.new Nodo(3);
        arbol.raiz.izquierdo.izquierdo = arbol.new Nodo(4);
        arbol.raiz.izquierdo.derecho = arbol.new Nodo(5);

        System.out.println("Altura del árbol es: " + arbol.altura(arbol.raiz));
        System.out.println("Profundidad del nodo con dato 5 es: " + arbol.profundidad(arbol.raiz.izquierdo.derecho, arbol.raiz));
    }
}

Explicación del Código

Este programa tiene dos funcionalidades principales: calcular la altura del árbol binario y la profundidad de un nodo específico. Ambas se implementan a través de métodos recursivos en la clase AlturaArbolBinario.

Clase Nodo

La clase interna Nodo representa cada nodo en el árbol binario, con referencias a sus hijos izquierdo y derecho. El constructor inicializa el valor de cada nodo y establece las referencias a los hijos como null inicialmente.

Método altura()

El método altura() calcula la altura del árbol a partir de un nodo específico, utilizando recursión. Calcula las alturas del subárbol izquierdo y derecho y devuelve el valor máximo más uno, representando el nivel de profundidad del nodo en el árbol.

Método profundidad()

El método profundidad() calcula la profundidad de un nodo desde la raíz del árbol. También usa recursión, buscando el nodo en los subárboles izquierdo y derecho hasta encontrarlo. Si el nodo se encuentra, retorna el nivel, de lo contrario devuelve -1.

Método main()

En el método main se crea un árbol binario y se llama a los métodos altura() y profundidad() para calcular la altura del árbol y la profundidad de un nodo específico (en este caso, el nodo con el valor 5).
