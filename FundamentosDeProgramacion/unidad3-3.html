Estructuras Iterativas en Programación

Explicación y ejemplos de las estructuras iterativas: repetir mientras, hasta, desde.

1. Estructura Iterativa "Repetir Mientras" (while)

En esta estructura, el código se repite mientras la condición sea verdadera.


let contador = 0;
while (contador < 5) {
    console.log("El contador es:", contador);
    contador++;
}

2. Estructura Iterativa "Repetir Hasta" (do-while)

En esta estructura, el código se ejecuta al menos una vez y luego se repite mientras la condición sea verdadera.


let contador = 0;
do {
    console.log("El contador es:", contador);
    contador++;
} while (contador < 5);

3. Estructura Iterativa "Repetir Desde" (for)

En esta estructura, el código se repite un número determinado de veces.


for (let contador = 0; contador < 5; contador++) {
    console.log("El contador es:", contador);
}
