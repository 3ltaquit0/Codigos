Modularidad en Programación

Aprende cómo dividir tu código en módulos para mejorar la organización, reutilización y mantenimiento.

1. ¿Qué es un módulo?

Un módulo es una unidad de código que agrupa funciones, variables y clases relacionadas. Su uso permite mejorar la estructura del código, hacerlo más reutilizable y fácil de mantener.

2. Ventajas de la Modularidad

  • Reutilización de código
  • Mejor mantenimiento
  • Organización y legibilidad
  • Encapsulamiento de detalles

3. Declaración y uso de módulos

Dependiendo del lenguaje de programación, los módulos se declaran de manera diferente. A continuación, algunos ejemplos de cómo declarar y usar módulos en varios lenguajes:

Ejemplo en Python


# math_utils.py
def suma(a, b):
    return a + b

def resta(a, b):
    return a - b
      

# main.py
import math_utils

resultado = math_utils.suma(3, 5)
print("El resultado de la suma es:", resultado)

Ejemplo en JavaScript (Node.js)


const mathUtils = require('./mathUtils');

let resultado = mathUtils.suma(3, 5);
console.log("El resultado de la suma es:", resultado);
