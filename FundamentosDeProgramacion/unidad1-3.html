Diseño de Algoritmos

El diseño de algoritmos es el proceso de crear una secuencia lógica de pasos para resolver un problema específico de forma eficiente.

Fases del Diseño de un Algoritmo

El diseño de un algoritmo involucra los siguientes pasos:

  • Entender el problema.
  • Planificar la solución.
  • Crear el algoritmo.
  • Verificar y mejorar el algoritmo.

Técnicas de Diseño de Algoritmos

  • Fuerza Bruta (Brute Force)
  • Divide y Vencerás (Divide and Conquer)
  • Programación Dinámica
  • Búsqueda y Ordenamiento

Ejemplo de Algoritmo: Encontrar el Número Máximo

A continuación se muestra el pseudocódigo para encontrar el número máximo en una lista de números:


Inicio
   Leer lista de números
   máximo = lista[0]
   Para cada número en la lista
      Si el número es mayor que máximo
         máximo = número
   Fin Para
   Escribir "El número máximo es", máximo
Fin

Implementación en Java


import java.util.List;

public class Maximo {
    public static int encontrarMaximo(List lista) {
        int maximo = lista.get(0);
        for (int numero : lista) {
            if (numero > maximo) {
                maximo = numero;
            }
        }
        return maximo;
    }

    public static void main(String[] args) {
        List lista = List.of(5, 3, 8, 6, 2, 7);
        int max = encontrarMaximo(lista);
        System.out.println("El número máximo es: " + max);
    }
}
