Elementos del Lenguaje de Programación

Conoce los elementos clave que conforman un lenguaje de programación.

1. Tipos de Datos

Los tipos de datos determinan qué tipo de valores pueden ser almacenados en una variable y qué operaciones se pueden realizar sobre esos valores. Los tipos básicos incluyen:

  • Enteros (int): Números enteros.
  • Decimales (float, double): Números con decimales.
  • Booleanos (bool): Valores de verdadero (true) o falso (false).
  • Cadenas (String): Secuencias de caracteres.
  • Caracteres (char): Un solo carácter.
        # Ejemplo en Python
        numero_entero = 10
        numero_decimal = 3.14
        es_verdadero = True
        saludo = "Hola Mundo"

2. Literales

Los literales son valores constantes que se utilizan directamente en el código, sin necesidad de ser almacenados en una variable. Algunos ejemplos son:

  • Entero: 42
  • Decimal: 3.14
  • Booleano: true
  • Cadena: "Hola"
        # Ejemplo en Python
        print(42)  # Literal entero
        print("Hola")  # Literal cadena

3. Constantes

Las constantes son valores que, una vez definidos, no pueden cambiar durante la ejecución del programa. Algunos lenguajes de programación permiten declarar constantes explícitamente. 

        # Ejemplo en C
        #define PI 3.14159  // Definición de constante
        printf("%f", PI);

En Python, no existe un mecanismo nativo para declarar constantes, pero por convención, se utilizan letras mayúsculas: 

        # Ejemplo en Python
        PI = 3.14159  # Convención de constante en Python

4. Variables

Las variables son contenedores que se utilizan para almacenar datos que pueden cambiar durante la ejecución del programa. En la mayoría de los lenguajes, es necesario declarar el tipo de dato que la variable almacenará. 

        # Ejemplo en Python
        edad = 25  # La variable 'edad' almacena un entero
        nombre = "Juan"  # La variable 'nombre' almacena una cadena

5. Identificadores

Los identificadores son los nombres que se asignan a las variables, funciones, clases y otros elementos del programa. Deben cumplir con ciertas reglas, como comenzar con una letra o guion bajo, y no pueden ser palabras reservadas del lenguaje. 

        # Ejemplo en Python
        nombre_usuario = "Ana"  # 'nombre_usuario' es un identificador válido
        _edad = 30  # '_edad' también es un identificador válido

6. Parámetros

Los parámetros son valores que se pasan a las funciones para que puedan realizar su tarea. Los parámetros permiten que las funciones trabajen con diferentes datos sin necesidad de cambiar su código interno. 

        # Ejemplo en Python
        def saludar(nombre):
            print(f"Hola, {nombre}!")

        saludar("Carlos")  # "Carlos" es el parámetro

7. Operadores

Los operadores son símbolos que realizan operaciones sobre los datos. Existen diferentes tipos de operadores, como los aritméticos, de comparación y lógicos:

  • Aritméticos: +, -, *, /, %
  • De comparación: ==, !=, <, >, <=, >=
  • Lógicos: and, or, not
        # Ejemplo en Python
        a = 5
        b = 3
        resultado = a + b  # Operador aritmético
        es_igual = a == b  # Operador de comparación
        es_verdadero = (a > b) and (b > 1)  # Operador lógico

8. Salida de Datos

La salida de datos es la manera en la que un programa muestra la información al usuario. La salida puede ser por pantalla, a través de un archivo, o mediante una interfaz gráfica. 

        # Ejemplo en Python
        print("¡Hola Mundo!")  # Imprime un mensaje en la consola
