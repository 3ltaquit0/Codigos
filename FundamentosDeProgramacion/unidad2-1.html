Introducción a la Programación: Conceptos Básicos

Aprende los fundamentos esenciales de la programación para comenzar a desarrollar tus propios programas.

1. ¿Qué es un Algoritmo?

Un algoritmo es un conjunto de pasos o instrucciones ordenadas que se siguen para resolver un problema o realizar una tarea específica.

Ejemplo de un algoritmo para sumar dos números:

        1. Leer dos números
        2. Sumar los dos números
        3. Mostrar el resultado

2. Lenguajes de Programación

Los lenguajes de programación son herramientas que usamos para escribir algoritmos y programas. Algunos lenguajes populares son:

  • Python: Fácil de aprender, ideal para principiantes y ampliamente utilizado.
  • Java: Orientado a objetos, utilizado en aplicaciones móviles y sistemas empresariales.
  • JavaScript: Lenguaje para el desarrollo web y la interacción en sitios web.
  • C++: Lenguaje poderoso para aplicaciones de alto rendimiento y videojuegos.

3. Variables y Tipos de Datos

Una variable es un espacio en memoria que se utiliza para almacenar información. Algunos tipos de datos comunes son:

  • Entero (int): Números enteros (ej. 1, -2, 3).
  • Decimal (float): Números con decimales (ej. 3.14, -5.7).
  • Cadena de texto (String): Secuencia de caracteres (ej. "Hola", "Programación").
  • Booleano (bool): Verdadero o falso (ej. true, false).

4. Estructuras de Control

Las estructuras de control permiten modificar el flujo de ejecución de un programa. Ejemplos:

Condicionales

Permiten tomar decisiones en función de una condición (usando `if`, `else`, `switch`). 

        Si (edad >= 18) entonces
            Escribir "Eres mayor de edad"
        Sino
            Escribir "Eres menor de edad"

Bucles (Iteración)

Permiten repetir un conjunto de instrucciones varias veces (usando `for`, `while`). 

        Para i desde 1 hasta 5 hacer
            Escribir "El valor de i es: ", i
        Fin Para

5. Funciones y Procedimientos

Las funciones son bloques de código que realizan una tarea específica y pueden devolver un valor. Los procedimientos son similares, pero no devuelven valores. 

        Función sumar(a, b)
            Retornar a + b
        Fin Función

6. Entrada y Salida

La entrada se refiere a la información que el programa recibe, y la salida es la información que el programa muestra. 

        Leer nombre
        Escribir "Hola, " + nombre

7. Conceptos de Orientación a Objetos (OOP)

La programación orientada a objetos organiza el código en objetos que tienen atributos y métodos.

  • Clase: Plantilla para crear objetos.
  • Objeto: Instancia de una clase.
  • Herencia: Permite que una clase herede atributos y métodos de otra clase.
  • Polimorfismo: Uso de una misma función de diferentes maneras.
  • Encapsulamiento: Oculta los detalles internos del objeto.
