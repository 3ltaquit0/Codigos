Diseño de Funciones

Las funciones son bloques de código que realizan tareas específicas y pueden ser reutilizadas dentro de un programa. Un diseño adecuado de funciones permite escribir código más modular, reutilizable y mantenible.

Características de una Función

  • Entrada (Parámetros): Datos que la función recibe para su procesamiento.
  • Proceso: La tarea que realiza la función con los parámetros.
  • Salida (Valor de Retorno): El resultado calculado por la función.
  • Encapsulamiento: La función oculta su implementación interna.

Tipos de Funciones

  • Funciones sin parámetros ni retorno.
  • Funciones con parámetros sin retorno.
  • Funciones sin parámetros con retorno.
  • Funciones con parámetros y retorno.

Ejemplo: Calcular el Área de un Círculo

El siguiente pseudocódigo describe una función que calcula el área de un círculo dado su radio:


Inicio
   Función CalcularÁrea(radio)
       área = 3.1416 * radio * radio
       Retornar área
   Fin Función
Fin

Implementación en Java


public class Calculadora {
    public static double calcularArea(double radio) {
        return Math.PI * radio * radio;
    }

    public static void main(String[] args) {
        double radio = 5.0;
        double area = calcularArea(radio);
        System.out.println("El área del círculo con radio " + radio + " es: " + area);
    }
}
