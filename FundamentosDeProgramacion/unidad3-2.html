Estructuras Selectivas en Programación

Explicación y ejemplos de las estructuras selectivas: simple, doble y múltiple.

1. Estructura Selectiva Simple

En una estructura simple, un bloque de código se ejecuta solo si la condición es verdadera.


let edad = prompt("¿Cuántos años tienes?");
if (edad >= 18) {
    alert("Eres mayor de edad.");
}

2. Estructura Selectiva Doble

En una estructura doble, un bloque de código se ejecuta si la condición es verdadera, y otro si es falsa.


let edad = prompt("¿Cuántos años tienes?");
if (edad >= 18) {
    alert("Eres mayor de edad.");
} else {
    alert("Eres menor de edad.");
}

3. Estructura Selectiva Múltiple

En una estructura múltiple, puedes verificar varias condiciones con múltiples bloques de código.


let edad = prompt("¿Cuántos años tienes?");
if (edad < 13) {
    alert("Eres un niño.");
} else if (edad < 18) {
    alert("Eres un adolescente.");
} else {
    alert("Eres un adulto.");
}
