Estructuras Lineales: Pilas

Una **pila** es una estructura de datos lineal que sigue el principio **LIFO** (Last In, First Out). Esto significa que el último elemento en ser agregado es el primero en salir. Es como una pila de platos: puedes poner un plato encima, pero siempre sacarás el de arriba.

¿Cómo funciona una pila?

Las pilas tienen dos operaciones principales:

  • Push: Agregar un elemento al final de la pila.
  • Pop: Eliminar el último elemento agregado.
También hay una operación adicional llamada peek, que devuelve el último elemento sin eliminarlo.

Ejemplo de una Pila en Java

A continuación, vamos a ver cómo se puede implementar una pila utilizando la clase Stack de Java. Este ejemplo muestra cómo agregar (push) y eliminar (pop) elementos de la pila. 


import java.util.Stack;

public class EjemploPila {
    public static void main(String[] args) {
        // Crear una nueva pila
        Stack pila = new Stack<>();

        // Agregar elementos a la pila
        pila.push(10);
        pila.push(20);
        pila.push(30);

        System.out.println("Pila después de agregar elementos: " + pila);

        // Eliminar el elemento superior de la pila (30)
        int elementoEliminado = pila.pop();
        System.out.println("Elemento eliminado: " + elementoEliminado);

        // Ver el elemento superior de la pila sin eliminarlo (20)
        int tope = pila.peek();
        System.out.println("Elemento en la cima de la pila: " + tope);

        // Mostrar la pila después de las operaciones
        System.out.println("Pila después de las operaciones: " + pila);
    }
}

En este código:

  • Se crea una pila de números enteros usando la clase Stack.
  • Se agregan tres elementos a la pila con el método push.
  • Se elimina el elemento superior de la pila con el método pop.
  • Finalmente, se muestra el elemento en la cima de la pila usando peek.
Como resultado, podemos ver cómo la pila funciona siguiendo el principio LIFO: el último elemento agregado es el primero en ser eliminado.

Aplicaciones de las Pilas

Las pilas son muy útiles en diferentes áreas de la programación, como:

  • Gestión de funciones: Se utilizan para gestionar las llamadas a funciones en los lenguajes de programación, especialmente en la implementación de la recursión.
  • Deshacer y rehacer: En aplicaciones como editores de texto, las pilas se utilizan para almacenar acciones previas y poder deshacerlas.
  • Algoritmos de recorrido: En algunos algoritmos de búsqueda y recorrido, como el recorrido en profundidad en grafos, se utilizan pilas para almacenar los nodos por visitar.

Conclusión

Las pilas son estructuras de datos simples pero poderosas, que siguen el principio LIFO para organizar y manipular los datos. Son esenciales para resolver diversos problemas de programación, especialmente en tareas relacionadas con la recursión, la gestión de funciones y la manipulación de datos de manera ordenada.
