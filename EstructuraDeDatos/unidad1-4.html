Ejemplos de Manejo de Memoria en Java

El manejo de memoria en Java es una parte fundamental para garantizar el rendimiento y la estabilidad de los programas. A continuación, se presentan algunos ejemplos clave para comprender cómo se maneja la memoria en Java.

1. Asignación de Memoria

En Java, la memoria se asigna automáticamente cuando se crean objetos utilizando el operador new. 


public class ManejoDeMemoria {
    public static void main(String[] args) {
        // Asignación de memoria para un objeto
        String saludo = new String("Hola, Mundo!");
        System.out.println(saludo);
    }
}

2. Liberación de Memoria con Garbage Collector

Java maneja la liberación de memoria de forma automática mediante el Garbage Collector, que elimina los objetos no utilizados. 


public class GarbageCollector {
    public static void main(String[] args) {
        // Crear objetos
        Object obj1 = new Object();
        Object obj2 = new Object();
        
        // Eliminación de objetos
        obj1 = null; // Hacemos que obj1 sea elegible para ser recolectado por el GC
        
        // Ejecutar el Garbage Collector manualmente
        System.gc(); // Solicitamos la recolección de basura, aunque el sistema lo hace automáticamente
    }
}

3. Ámbito de las Variables y Memoria

Las variables locales se almacenan en la pila y son eliminadas cuando el método termina su ejecución. 


public class VariablesYMemoria {
    public static void main(String[] args) {
        // Variables locales almacenadas en la pila
        int numero = 10;
        System.out.println("Número: " + numero);
    }
}

4. Ejemplo de Fugas de Memoria

Una fuga de memoria ocurre cuando un objeto ya no es necesario, pero todavía hay referencias a él, impidiendo que el Garbage Collector lo elimine. 


import java.util.ArrayList;

public class FugasDeMemoria {
    public static void main(String[] args) {
        ArrayList<Object> lista = new ArrayList<>();
        
        // Se crea un objeto pero no se elimina adecuadamente
        for (int i = 0; i < 1000000; i++) {
            lista.add(new Object());
        }

        System.out.println("Se han agregado 1 millón de objetos a la lista.");
    }
}

Conclusión

En Java, la gestión de la memoria se maneja de manera automática mediante la asignación de memoria dinámica y el Garbage Collector. Sin embargo, es importante comprender cómo funciona para evitar problemas como las fugas de memoria.
