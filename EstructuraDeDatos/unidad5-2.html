Algoritmo de Ordenamiento por Burbuja

El algoritmo de ordenamiento por burbuja (Bubble Sort) es uno de los algoritmos más sencillos de entender y de implementar. Su funcionamiento consiste en comparar elementos adyacentes en una lista o arreglo y, si están en el orden incorrecto, intercambiarlos. Este proceso se repite varias veces hasta que todos los elementos estén ordenados de acuerdo al criterio establecido (generalmente ascendente o descendente).

1. ¿Cómo funciona el algoritmo de burbuja?

La idea detrás del ordenamiento por burbuja es simple: se compara cada par de elementos adyacentes en la lista y se intercambian si no están en el orden deseado. Al final de cada pasaje, el elemento más grande "burbujea" hacia el final de la lista, de ahí su nombre.

El proceso se repite varias veces, y en cada pasada se va moviendo el siguiente elemento más grande hacia su lugar correspondiente. El algoritmo termina cuando ya no es necesario realizar más intercambios.

2. Paso a paso del algoritmo de burbuja

  1. Comienza en el primer elemento de la lista.
  2. Compara el elemento actual con el siguiente.
  3. Si el elemento actual es mayor que el siguiente, se intercambian.
  4. Continúa hasta el final de la lista.
  5. Repite el proceso para los elementos restantes, excluyendo los elementos ya ordenados.
  6. El proceso termina cuando no se necesitan más intercambios.

3. Ejemplo de ordenamiento por burbuja

Supongamos que tenemos la siguiente lista de números desordenados: [5, 3, 8, 4, 2].

Durante el primer paso, comparamos 5 con 3, como 5 es mayor que 3, los intercambiamos. La lista se convierte en [3, 5, 8, 4, 2].

Luego, comparamos 5 con 8. Como 5 es menor que 8, no se realiza ningún cambio. Continuamos con 8 y 4, y como 8 es mayor que 4, los intercambiamos, lo que da como resultado [3, 5, 4, 8, 2].

Finalmente, comparamos 8 con 2, los intercambiamos, obteniendo la lista [3, 5, 4, 2, 8].

El proceso se repite para las siguientes pasadas hasta que la lista está completamente ordenada.

4. Implementación en Java


public class BubbleSort {
    public static void bubbleSort(int[] array) {
        int n = array.length;
        for (int i = 0; i < n - 1; i++) {
            for (int j = 0; j < n - i - 1; j++) {
                if (array[j] > array[j + 1]) {
                    // Intercambiar los elementos
                    int temp = array[j];
                    array[j] = array[j + 1];
                    array[j + 1] = temp;
                }
            }
        }
    }

    public static void main(String[] args) {
        int[] array = {5, 3, 8, 4, 2};
        bubbleSort(array);
        System.out.println("Lista ordenada: ");
        for (int num : array) {
            System.out.print(num + " ");
        }
    }
}

5. Complejidad del algoritmo de burbuja

La complejidad temporal del algoritmo de ordenamiento por burbuja en el peor caso es O(n²), donde n es el número de elementos en la lista. Esto se debe a que en cada pasada se compara cada par de elementos, lo que resulta en un número de comparaciones cuadrático.

En el mejor de los casos, si la lista ya está ordenada, el algoritmo puede terminar en O(n) si se optimiza para detectar que no hubo intercambios en una pasada.

6. Ventajas y desventajas del algoritmo de burbuja

Ventajas

  • Fácil de entender y de implementar.
  • No requiere memoria adicional, ya que es un algoritmo de ordenamiento en el lugar (in-place).

Desventajas

  • Es ineficiente para listas grandes, ya que tiene una complejidad O(n²).
  • No es adecuado para conjuntos de datos que no sean pequeños o medianos.

7. Conclusión

El algoritmo de ordenamiento por burbuja es una excelente opción para aprender sobre algoritmos de ordenamiento debido a su simplicidad. Sin embargo, para listas más grandes, existen algoritmos más eficientes, como Quick Sort o Merge Sort, que tienen una complejidad mejor (O(n log n)) y son mucho más rápidos en la práctica.
