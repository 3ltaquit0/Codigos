Tipos de Datos Abstractos (TDA)

Un Tipo de Dato Abstracto (TDA) es una estructura de datos que define qué operaciones se pueden realizar sobre los datos, pero no cómo se implementan internamente. Son fundamentales en la programación porque nos ayudan a organizar y manipular información de manera eficiente.

1. Lista

Una lista es una colección ordenada de elementos donde podemos agregar, eliminar y buscar datos fácilmente. 


import java.util.ArrayList;

public class EjemploLista {
    public static void main(String[] args) {
        ArrayList lista = new ArrayList<>();
        lista.add("Manzana");
        lista.add("Banana");
        lista.add("Cereza");

        for (String fruta : lista) {
            System.out.println(fruta);
        }
    }
}

2. Pila (Stack)

Una pila sigue el principio LIFO (Last In, First Out), es decir, el último en entrar es el primero en salir. 


import java.util.Stack;

public class EjemploPila {
    public static void main(String[] args) {
        Stack pila = new Stack<>();
        pila.push(10);
        pila.push(20);
        pila.push(30);

        System.out.println("Elemento eliminado: " + pila.pop()); // Sale 30
    }
}

3. Cola (Queue)

Una cola sigue el principio FIFO (First In, First Out), el primero en entrar es el primero en salir. 


import java.util.LinkedList;
import java.util.Queue;

public class EjemploCola {
    public static void main(String[] args) {
        Queue cola = new LinkedList<>();
        cola.add("A");
        cola.add("B");
        cola.add("C");

        System.out.println("Elemento eliminado: " + cola.poll()); // Sale "A"
    }
}

4. Conjunto (Set)

Un conjunto es una colección de elementos únicos, es decir, no permite duplicados. 


import java.util.HashSet;

public class EjemploConjunto {
    public static void main(String[] args) {
        HashSet conjunto = new HashSet<>();
        conjunto.add(1);
        conjunto.add(2);
        conjunto.add(2); // No se agregará porque ya existe

        System.out.println(conjunto);
    }
}

5. Diccionario (Map)

Un diccionario almacena pares clave-valor, permitiendo encontrar datos rápidamente. 


import java.util.HashMap;

public class EjemploDiccionario {
    public static void main(String[] args) {
        HashMap edades = new HashMap<>();
        edades.put("Juan", 25);
        edades.put("María", 30);

        System.out.println("Edad de Juan: " + edades.get("Juan"));
    }
}

Conclusión

Los Tipos de Datos Abstractos nos ayudan a manejar datos de manera estructurada y eficiente. Dependiendo del problema que enfrentemos, elegiremos la estructura de datos más adecuada:

  • Listas: Si necesitamos mantener un orden específico.
  • Pilas: Para estructuras donde el último en entrar debe salir primero.
  • Colas: Para gestionar procesos en orden de llegada.
  • Conjuntos: Cuando los valores no pueden repetirse.
  • Diccionarios: Para almacenar datos asociados a claves específicas.
