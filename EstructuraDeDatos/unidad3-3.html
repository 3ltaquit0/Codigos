Operaciones Básicas de una Pila

Una pila es una estructura de datos que sigue el principio **LIFO** (Last In, First Out), lo que significa que el último elemento agregado es el primero en ser eliminado. Las operaciones básicas que se realizan sobre una pila son: **push**, **pop**, **peek**, y **isEmpty**.

Operaciones Principales de una Pila

1. Operación Push

La operación **push** agrega un elemento a la cima de la pila. Este es el único lugar donde se pueden agregar elementos en una pila.

En Java, la operación **push** se realiza utilizando el método push(). A continuación, se muestra un ejemplo: 


import java.util.Stack;

public class EjemploPush {
    public static void main(String[] args) {
        Stack pila = new Stack<>();
        
        // Agregar elementos a la pila
        pila.push(10);
        pila.push(20);
        pila.push(30);

        System.out.println("Pila después de Push: " + pila);
    }
}

Salida: [10, 20, 30] (El último elemento agregado es 30 y está en la cima).

2. Operación Pop

La operación **pop** elimina el elemento que está en la cima de la pila. Después de realizar esta operación, la pila se reduce en tamaño y el elemento eliminado ya no está disponible.

En Java, la operación **pop** se realiza utilizando el método pop(). A continuación, se muestra un ejemplo: 


import java.util.Stack;

public class EjemploPop {
    public static void main(String[] args) {
        Stack pila = new Stack<>();
        
        // Agregar elementos a la pila
        pila.push(10);
        pila.push(20);
        pila.push(30);

        // Eliminar el elemento superior de la pila
        System.out.println("Elemento eliminado: " + pila.pop()); // 30

        System.out.println("Pila después de Pop: " + pila);
    }
}

Salida: Elemento eliminado: 30
Pila después de Pop: [10, 20] (El último elemento agregado fue 30, por lo que es el primero en ser eliminado).

3. Operación Peek

La operación **peek** permite ver el elemento que está en la cima de la pila sin eliminarlo. Es útil cuando quieres acceder al elemento sin modificar la pila.

En Java, la operación **peek** se realiza utilizando el método peek(). A continuación, se muestra un ejemplo: 


import java.util.Stack;

public class EjemploPeek {
    public static void main(String[] args) {
        Stack pila = new Stack<>();
        
        // Agregar elementos a la pila
        pila.push(10);
        pila.push(20);
        pila.push(30);

        // Ver el elemento en la cima de la pila
        System.out.println("Elemento en la cima de la pila: " + pila.peek()); // 30
    }
}

Salida: Elemento en la cima de la pila: 30 (El último elemento agregado es 30).

4. Operación isEmpty

La operación **isEmpty** permite verificar si la pila está vacía o no. Devuelve un valor booleano: true si la pila está vacía, y false si contiene elementos.

En Java, la operación **isEmpty** se realiza utilizando el método isEmpty(). A continuación, se muestra un ejemplo: 


import java.util.Stack;

public class EjemploIsEmpty {
    public static void main(String[] args) {
        Stack pila = new Stack<>();
        
        // Verificar si la pila está vacía
        System.out.println("¿Está la pila vacía? " + pila.isEmpty()); // true
        
        // Agregar elementos a la pila
        pila.push(10);
        
        // Verificar nuevamente
        System.out.println("¿Está la pila vacía? " + pila.isEmpty()); // false
    }
}

Salida: ¿Está la pila vacía? true
¿Está la pila vacía? false (Después de agregar un elemento, la pila ya no está vacía).

Resumen de Operaciones

  • Push: Agrega un elemento a la cima de la pila.
  • Pop: Elimina el elemento superior de la pila y lo devuelve.
  • Peek: Devuelve el elemento en la cima de la pila sin eliminarlo.
  • isEmpty: Verifica si la pila está vacía.

Conclusión

Las pilas son estructuras de datos simples pero muy poderosas. Las operaciones básicas como **push**, **pop**, **peek**, y **isEmpty** permiten manipular los elementos de manera eficiente, siguiendo el principio LIFO. Estas operaciones son fundamentales para comprender cómo funcionan las pilas en programación y se usan comúnmente en algoritmos y estructuras de datos.
