Ejemplos de Tipos de Datos Abstractos (TDA)

Los Tipos de Datos Abstractos (TDA) permiten organizar y manipular datos de manera eficiente en los programas. A continuación, se presentan ejemplos en Java de algunos TDA comunes: Lista, Pila, Cola, Conjunto y Diccionario.

1. Lista Enlazada

Una Lista Enlazada es una estructura donde cada nodo apunta al siguiente nodo, formando una secuencia dinámica. 


class Nodo {
    int dato;
    Nodo siguiente;

    public Nodo(int dato) {
        this.dato = dato;
        this.siguiente = null;
    }
}

class ListaEnlazada {
    private Nodo cabeza;

    public void agregar(int dato) {
        Nodo nuevo = new Nodo(dato);
        if (cabeza == null) {
            cabeza = nuevo;
        } else {
            Nodo temp = cabeza;
            while (temp.siguiente != null) {
                temp = temp.siguiente;
            }
            temp.siguiente = nuevo;
        }
    }

    public void mostrar() {
        Nodo temp = cabeza;
        while (temp != null) {
            System.out.print(temp.dato + " -> ");
            temp = temp.siguiente;
        }
        System.out.println("null");
    }
}

public class EjemploListaEnlazada {
    public static void main(String[] args) {
        ListaEnlazada lista = new ListaEnlazada();
        lista.agregar(1);
        lista.agregar(2);
        lista.agregar(3);
        lista.mostrar();
    }
}

2. Pila (Stack)

Una Pila sigue el principio LIFO (Last In, First Out). Se usa para gestionar funciones y operaciones reversibles. 


import java.util.Stack;

public class EjemploPila {
    public static void main(String[] args) {
        Stack pila = new Stack<>();
        pila.push(10);
        pila.push(20);
        pila.push(30);

        System.out.println("Elemento eliminado: " + pila.pop()); // 30
        System.out.println("Tope de la pila: " + pila.peek()); // 20
    }
}

3. Cola (Queue)

Una Cola sigue el principio FIFO (First In, First Out), útil para gestionar procesos en orden. 


import java.util.LinkedList;
import java.util.Queue;

public class EjemploCola {
    public static void main(String[] args) {
        Queue cola = new LinkedList<>();
        cola.add("A");
        cola.add("B");
        cola.add("C");

        System.out.println("Elemento eliminado: " + cola.poll()); // "A"
        System.out.println("Frente de la cola: " + cola.peek()); // "B"
    }
}

4. Conjunto (Set)

Un Conjunto almacena elementos únicos sin orden específico. 


import java.util.HashSet;

public class EjemploConjunto {
    public static void main(String[] args) {
        HashSet conjunto = new HashSet<>();
        conjunto.add(1);
        conjunto.add(2);
        conjunto.add(2); // No se agrega porque ya existe

        System.out.println(conjunto);
    }
}

5. Diccionario (Map)

Un Diccionario almacena pares clave-valor para una rápida recuperación. 


import java.util.HashMap;

public class EjemploDiccionario {
    public static void main(String[] args) {
        HashMap edades = new HashMap<>();
        edades.put("Juan", 25);
        edades.put("María", 30);

        System.out.println("Edad de Juan: " + edades.get("Juan"));
    }
}

Conclusión

Los TDA permiten manejar datos de manera estructurada y eficiente. Cada estructura tiene sus ventajas y aplicaciones en la programación.
