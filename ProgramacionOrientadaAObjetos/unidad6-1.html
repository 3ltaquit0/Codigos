Flujos y Archivos en Programación

Flujos

Un flujo (o "stream" en inglés) es una secuencia de datos que se transmite de un lugar a otro, como una secuencia de bytes o caracteres. Los flujos se utilizan para leer o escribir datos desde y hacia dispositivos o archivos, ya sea en memoria o en dispositivos de almacenamiento.

Existen dos tipos principales de flujos:

  • Flujos de entrada (Input Streams): Permiten leer datos desde una fuente (por ejemplo, desde un archivo o la entrada estándar).
  • Flujos de salida (Output Streams): Permiten escribir datos a un destino (por ejemplo, un archivo o la salida estándar).

Los flujos permiten realizar operaciones de lectura y escritura de manera secuencial, facilitando el acceso eficiente a los datos.

Archivos

Un archivo es una unidad de almacenamiento en un dispositivo, como un disco duro, donde los datos se guardan de manera persistente. Los archivos pueden contener diversos tipos de datos, como texto, imágenes, videos, entre otros.

Los archivos pueden ser de diferentes tipos:

  • Archivos de texto: Contienen datos en formato legible por el ser humano.
  • Archivos binarios: Contienen datos en formato no legible por humanos (por ejemplo, imágenes o videos).

Los archivos se utilizan para almacenar datos de forma que persistan más allá de la ejecución de un programa.

Flujos y Archivos en conjunto

Los flujos y los archivos trabajan juntos para permitir la manipulación eficiente de datos. Al trabajar con archivos, se usan flujos para leer o escribir datos de manera secuencial. Los flujos permiten interactuar con los archivos de una manera controlada.

Operaciones comunes con archivos:

  • Abrir un archivo para leer o escribir.
  • Leer datos desde un archivo usando flujos de entrada.
  • Escribir datos a un archivo usando flujos de salida.
  • Cerrar el archivo después de realizar las operaciones.

Ejemplo de código en Java


import java.io.*;

public class EjemploArchivo {
    public static void main(String[] args) {
        try (FileOutputStream archivoSalida = new FileOutputStream("ejemplo.txt")) {
            String texto = "Hola, este es un ejemplo de escritura en archivo.";
            archivoSalida.write(texto.getBytes());
        } catch (IOException e) {
            e.printStackTrace();
        }

        try (FileInputStream archivoEntrada = new FileInputStream("ejemplo.txt")) {
            int contenido;
            while ((contenido = archivoEntrada.read()) != -1) {
                System.out.print((char) contenido);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
