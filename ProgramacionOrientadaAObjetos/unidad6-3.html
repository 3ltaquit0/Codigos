Operaciones Básicas y Tipos de Acceso en Archivos

Operaciones Básicas en Archivos

Las operaciones básicas en archivos son esenciales para interactuar con ellos. A continuación, se describen las principales operaciones que se realizan en archivos:

  • Abrir: Antes de realizar cualquier operación en un archivo, es necesario abrirlo. Dependiendo del tipo de archivo (texto o binario) y del modo de acceso, se abrirá en un modo específico (lectura, escritura, etc.).
  • Leer: Esta operación permite extraer información del archivo. En archivos de texto, se lee línea por línea o carácter por carácter. En archivos binarios, se lee byte por byte.
  • Escribir: Permite insertar información en el archivo. En archivos de texto, se escribe como texto plano. En archivos binarios, se escriben secuencias de bytes.
  • Cerrar: Después de realizar las operaciones necesarias, se debe cerrar el archivo para liberar los recursos del sistema.

Tipos de Acceso a Archivos

El acceso a los archivos se puede clasificar en diferentes tipos, que determinan cómo y en qué orden se accede a los datos almacenados en un archivo:

  • Acceso Secuencial: En este tipo de acceso, los datos se leen o escriben de manera secuencial, es decir, en el orden en que se encuentran en el archivo. El acceso secuencial es común en archivos de texto, como los archivos de log, donde los datos se procesan en el orden en que fueron escritos.
  • Acceso Directo: Este tipo de acceso permite leer o escribir en cualquier parte del archivo sin necesidad de procesar los datos secuencialesmente. Los archivos binarios generalmente permiten acceso directo, lo que facilita la búsqueda rápida de datos en grandes archivos.
  • Acceso Aleatorio: Permite saltar entre las partes del archivo y acceder a cualquier ubicación de manera directa. Este tipo de acceso es útil para bases de datos y otros sistemas que necesitan leer y escribir datos de manera no secuencial.

Ejemplos de Operaciones en Archivos

A continuación, se presenta un ejemplo de cómo realizar algunas de las operaciones básicas en archivos en Java:

Operación de Lectura y Escritura en Archivos de Texto:


import java.io.*;

public class EjemploArchivoTexto {
    public static void main(String[] args) {
        // Escritura en archivo de texto
        try (BufferedWriter escritor = new BufferedWriter(new FileWriter("archivo.txt"))) {
            escritor.write("Hola, este es un archivo de texto.");
        } catch (IOException e) {
            e.printStackTrace();
        }

        // Lectura de archivo de texto
        try (BufferedReader lector = new BufferedReader(new FileReader("archivo.txt"))) {
            String linea;
            while ((linea = lector.readLine()) != null) {
                System.out.println(linea);
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}

Operación de Lectura y Escritura en Archivos Binarios:


import java.io.*;

public class EjemploArchivoBinario {
    public static void main(String[] args) {
        // Escritura en archivo binario
        try (FileOutputStream archivoSalida = new FileOutputStream("archivo.bin")) {
            byte[] datos = {1, 2, 3, 4, 5};
            archivoSalida.write(datos);
        } catch (IOException e) {
            e.printStackTrace();
        }

        // Lectura de archivo binario
        try (FileInputStream archivoEntrada = new FileInputStream("archivo.bin")) {
            int contenido;
            while ((contenido = archivoEntrada.read()) != -1) {
                System.out.print(contenido + " ");
            }
        } catch (IOException e) {
            e.printStackTrace();
        }
    }
}
